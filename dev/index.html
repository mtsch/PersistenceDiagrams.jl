PersistenceDiagrams.jl

This package provides the PersistenceInterval and PersistenceDiagram types as well as some functions for working with them. If you want to compute persistence diagrams, please see Ripserer.jl. For examples and tutorials, see the Ripserer.jl docs.

Overview

This package currently supports the following:

  • persistence diagram plotting
  • bottleneck and Wasserstein matching and distance computation
  • various vectorization methods including persistence images, betti curves, landscapes, and more (see Vectorization for full list)
  • integration with MLJ.jl.
